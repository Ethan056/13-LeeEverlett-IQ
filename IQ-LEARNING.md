# 12/12/2568

ขั้นตอนของโปรแกรม ตื่นมาเล่นเกมกัน!

1. ตั้งนาฬิกาปลุกนอนงีบหลับ
2. เมื่อถึงเวลา นาฬิกาจะปลุก
3. หากผู้ใช้งานไม่ตื่นมาปิด โปรแกรมจะค่อยๆเพิ่มระดับความดังของเสียงเรื่อยๆจนกว่าผู้ใช้จะปิดโปรแกรม

ผลการวิเคราห์ตนเองกับโจทย์

1. ไม่ยากเลย ไหวชิว
2. ชอบผังงานมากกว่า เพราะสามารถเห็นภาพรวมของโปรแกรมได้ หากเกิดข้อผิดพลาดจะสามารถแก้ไขได้รวดเร็ว
3. ยากที่สุดคือขั้นที่ 3 เพราะผู้ใช้อาจจะหลับลึกจนเสียงไม่ดังพอที่จะปลุก แก้โดยให้โปรแกรมเพิ่มเสียงให้ดังขึ้นอีกเรื่อยๆจนเกือบถึงลิมิต
